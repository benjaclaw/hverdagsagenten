# Hverdagsagenten

> Overvåk Finn.no-annonser og matvarepriser — alt i én app.

## Tech Stack
- Expo SDK 52+ / Expo Router
- TypeScript / NativeWind
- Supabase (auth + database)
- React Query / Zustand
- Kassal.app API

## Getting Started

```bash
npm install
cp .env.example .env.local  # Fill in values
npx expo start
```

## Features
- 🔍 Finn.no-søk: opprett, rediger, slett, push-varsler
- 🛒 Matvaretilbud: overvåk priser, se historikk
- 📊 Dashboard med alle aktive overvåkninger
- 🔔 Push-varsler ved nye treff/tilbud
- 🌙 Mørk/lys modus
